# Job Board

A full-stack job board built on Next.js 14 App Router with end-to-end type safety, Server Actions, and a real Postgres database.

## 🚧 Status

In active development — Days 11-20 of a 30-day frontend sprint.

## 🛠 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Database:** Postgres (Neon free tier)
- **ORM:** Drizzle
- **State:** Zustand (client-side filter state)
- **Styling:** Tailwind CSS
- **Validation:** Zod
- **Testing:** Jest + React Testing Library
- **CI/CD:** GitHub Actions

## 🎯 Planned Features

- Server-rendered job listing pages with filters
- Dynamic job detail routes
- Debounced URL-param search (shareable links)
- Post-a-job form using Next.js Server Actions + Zod validation
- Save-jobs functionality with Zustand
- Optional auth via NextAuth
- 5–10 component/server tests
- GitHub Actions CI running tests + typecheck on push
